# Privacy Policy

**Last Updated: November 17, 2025**

## Overview

Claims Data Exporter is an open-source browser extension that helps users backup and export their own insurance claims data from systems they have authorized access to. We are committed to protecting your privacy.

## Data Collection

**We do not collect, store, or transmit any user data.**

This extension:
- ✅ Does NOT collect personal information
- ✅ Does NOT track user behavior
- ✅ Does NOT transmit data to external servers
- ✅ Does NOT use analytics or tracking services
- ✅ Does NOT sell or share data with third parties

## How the Extension Works

1. **Local Processing**: All data processing happens entirely within your browser
2. **Your Session**: Uses your existing authenticated browser session to access claims data
3. **Your Control**: You choose what data to export and when
4. **Your Computer**: Exported data is downloaded directly to your computer

## Permissions Explained

### activeTab
Used to interact with the claims management platform tab when you click the extension icon. This allows the extension to read your existing session and make API calls on your behalf.

### host_permissions (claimwizard.com)
Required to make API requests to the claims platform you're already logged into. This enables the extension to retrieve your data using your existing authenticated session.

### storage
Used to store only UI preferences and processing state within your browser. No claims data or personal information is stored.

## No Remote Code

This extension does not load or execute any remote code. All functionality is contained within the extension package, which you can audit on our [GitHub repository](https://github.com/jacoboressie/claims-data-exporter).

## Open Source

Our entire codebase is open source and available for review. You can verify exactly what this extension does by reviewing the code at:

https://github.com/jacoboressie/claims-data-exporter

## Data Security

- All data remains in your browser during the export process
- No data is sent to external servers
- No credentials are stored or transmitted
- Exported files are saved directly to your local computer

## Third-Party Services

This extension does NOT use:
- Analytics services
- Crash reporting services
- Advertising networks
- Third-party APIs (except the claims platform you're already using)

## Your Rights

You have complete control:
- Choose what data to export
- Choose when to export it
- All data remains on your device
- Uninstall the extension at any time
